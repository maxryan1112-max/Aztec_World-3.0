#Aztec World
Aztec World is a first-person educational exploration game set in an Aztec-inspired environment, designed to run as an embedded application inside a desktop-style portfolio website. The game is built as a vertical slice prototype, meaning it covers one complete and polished level rather than a full game experience. Players explore a starting area, speak with a merchant NPC, purchase tools from her shop, and earn knowledge points that reflect their learning progress.

##Navigation and Controls

Players interact with the game using a keyboard and mouse. The following controls are used throughout the level:
Movement: Use W, A, S, D to walk through the environment
Look around: Move the mouse to control the camera direction
Interact: Press E when near the merchant to begin the dialogue
Select dialogue options: Left-click on a response to advance the conversation
Navigate the shop: Left click to select an item, then click Confirm Purchase to buy it
Close the shop: Click the X button in the top corner of the shop panel
When the player is close enough to the merchant to interact, a prompt will appear at the bottom of the screen. This is the only instruction the game provides, so players are encouraged to explore freely and approach the merchant at their own pace.

##System Requirements

Aztec World is designed to run in modern web browsers including Chrome, Firefox, Safari, and Edge. No additional installation is required and the game loads directly through the portfolio website. For the best experience, use a desktop or laptop with a keyboard and mouse. The game is not currently optimized for mobile or touchscreen devices
